Como fazer para Rodar
Pra você conseguir testar o jogo na sua máquina, siga estes passos. É bem rápido.

1. O que precisa ter
Você só precisa ter o Node.js instalado, que já vem com o npm. Se não tiver, é só instalar a versão mais recente.

2. Entrando na pasta
Abra seu terminal (ou Prompt de Comando, Powershell, etc.) e navegue até a pasta principal do projeto:

cd sua-pasta-do-jogo

3. Instalando as coisas
Agora, rode o comando para baixar todas as dependências do React e do Vite:

npm install

4. Ligando o servidor
Com as dependências instaladas, é só iniciar o ambiente de desenvolvimento:

npm run dev

5. Jogando
O comando acima vai te dar um link local, tipo: http://localhost:5173/.


app rodando
